<h1 align="center">Smart Optimal Salves - SKSE</h1>

Just like my [Press E to Heal Followers SKSE](https://www.nexusmods.com/skyrimspecialedition/mods/171733) i moved the Papyrus script for the inventory scanning function to SKSE, making the process instant, and it should eliminate the lag. You still need the original mod, and it will function the same way how original mod did.

## Requirements
- [Address Library for SKSE Plugin](https://www.nexusmods.com/skyrimspecialedition/mods/32444)
- [Skyrim Script Extender (SKSE64)](https://www.nexusmods.com/skyrimspecialedition/mods/30379)
- [Smart Optimal Salves - Optimal Potion Hotkey MCM](https://www.nexusmods.com/skyrimspecialedition/mods/42402)

## Compatible:
**Should work out of the box. If something does not let me know**
- [Requiem - The Roleplaying Overhaul](https://www.nexusmods.com/skyrimspecialedition/mods/60888)
- [Apothecary - An Alchemy Overhaul](https://www.nexusmods.com/skyrimspecialedition/mods/52130)
- [Alchemy Potions and Food Adjustments](https://www.nexusmods.com/skyrimspecialedition/mods/5877?tab=description)
- [CACO](https://www.nexusmods.com/skyrimspecialedition/mods/19924)

## Not Compatible:
- [Smart Optimal Salves - A Patch for All Potions](https://www.nexusmods.com/skyrimspecialedition/mods/48479) (Built into the mod)

If any mod touches the "zzzPotionHotkeyScript.psc" script is not **Not Compatible.**
